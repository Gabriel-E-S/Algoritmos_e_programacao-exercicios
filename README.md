Hello! 👋 Welcome to the repository dedicated to the activities and exercises of the **Algorithms and Programming** course, taken at the **State University of Ponta Grossa (UEPG)**.

## 🎯 Repository Objective

The purpose of this space is to document my learning in building logical and structured algorithms, serving both as a personal portfolio and as reference material for other students who are on the same path.

## 📂 Content

The solutions to the activities proposed throughout the semester are organized in two ways:

* **C Language:** Real, compilable, and functional implementation of the proposed algorithms.
* **Pseudocode:** Step-by-step logical structuring to facilitate the understanding of data flow and the math behind the code.

### 📝 Note on Pseudocode
The pseudocode format used in the notes of this repository was developed exclusively by the course professor. It has its own didactic syntax and is strongly inspired by the structures of **C** and **Pascal** languages.

## 💻 How to run the C codes

1. Clone this repository to your machine:
   ```bash
   git clone https://github.com/Gabriel-E-S/Algoritmos_e_programacao-exercicios.git
   ```

# Algoritmos e Programação | UEPG -- (Texto em Português)

Olá! 👋 Bem-vindo ao repositório dedicado às atividades e exercícios da disciplina de **Algoritmos e Programação**, cursada na **Universidade Estadual de Ponta Grossa (UEPG)**.

## 🎯 Objetivo do Repositório

O intuito deste espaço é documentar o meu aprendizado na construção de algoritmos lógicos e estruturados, servindo tanto como um portfólio pessoal quanto como material de consulta para outros estudantes que estão trilhando o mesmo caminho.

## 📂 Conteúdo

As resoluções das atividades propostas ao longo do semestre estão organizadas de duas formas:
